Description

The focal point of running a simulation is that once we have a reasonable model, we can study the phenomenon or system of interest under conditions that are impossible at the moment. For example, what happens if clients start coming twice as often? We'd love to run this experiment, but it's not clear how to make them come more frequently. Or another example: what happens if we hire one more barber? What happens to our profit margin? We'd like to evaluate the risk before we take it.

In this stage, assume that one of the masters fell sick. How will the values that we calculated in the previous stage change?

Objectives

Write a program that takes as input four numbers (separated by one space):

  • \(s\), the average time in minutes between successive visitors,
  • \(a\), \(b\), the lower and upper boundaries for the service time distribution,
  • and the number of barbers \(k\)

for \(k-1\) and \(k\) barbers, it should estimate

  • the 95% quantile of the maximum waiting time,
  • the average closing time delay,
  • the probability that maximum number of visitors in line is greater that 5,

Use 100000 Monte-Carlo simulations, and print the differences between the corresponding metrics on separate lines.
