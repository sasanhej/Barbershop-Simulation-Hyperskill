Description

Finally, let's simulate the data we took as input in the previous two stages! We need to simulate the arrival times of clients and the times it takes our barbers to serve them.

If we assume that clients come independently at a fixed frequency (\(r\) people per hour on average), we can simulate the time between successive visitors as the exponential random variable. So that the average time between two visitors is \(60 / r\) minutes. This is a rare case in real life: for example, clients opt for evening hours, but it is a reasonable starting point.

The serving time is a bit different. Some come for a shave or a quick trim, while others want an elaborate haircut. These services' exact proportions and distributions depend on the barbershop and a particular barber. It is best to choose the appropriate distribution based on specific data. In this project, we will use the oversimplified model, the uniform distribution, from \(a\) to \(b\) minutes for every barber.

Since we treat the input data as random, all the metrics we calculate in the previous stage also become random. A natural step would be to investigate the behavior of the distributions of metrics depending on the distributions of the input data using histograms/bar plots and various location/scale parameters. In this stage, focus on the following parameters:

  • 95% quantile of the maximum waiting time â€” it shows that the maximum waiting time exceeds it on only 5% of days;
  • the average closing time delay;
  • the probability that the maximum number of visitors in line is more than 5. Imagine there are only five seats in the waiting area, so we want to know the probability that at least one client will not have enough seating today.

Strictly speaking, all the parameters we want to calculate are uniquely determined by the parameters of the arrival and service time distributions, so it is tempting to derive formulas that would allow us to calculate them accurately. Unfortunately, the model we have built is quite complex, so it does not seem possible to derive these formulas. Instead, we will estimate the required values using the Monte Carlo method: simulate one day in the barbershop many times, calculate all the metrics for each day, and then aggregate the obtained data.

Objectives

Write a program that takes as input four numbers (separated by one space):

  • \(s\), the average time in minutes between successive visitors,
  • \(a\), \(b\), the lower and upper bounds for service time distribution,
  • and the number of barbers \(k\)

and estimates:

  • the 95% quantile of the maximum waiting time,
  • the average closing time delay,
  • the probability that the maximum number of visitors in line is greater than 5.

Use 10000 Monte-Carlo simulations and print them on a separate line.
